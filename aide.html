Questions fréquentes

Pour prendre rendez-vous avec un médecin, suivez ces étapes simples :

  1. Sur la page d'accueil, cliquez sur "Je prends rendez-vous"
  2. Sélectionnez le type de praticien souhaité (généraliste, cardiologue, ostéopathe)
  3. Choisissez une date disponible dans le calendrier
  4. Sélectionnez l'horaire qui vous convient
  5. Confirmez votre rendez-vous

Pour commander vos médicaments, vous avez deux options :

  • Livraison à domicile : Sélectionnez cette option et indiquez votre adresse de livraison. Vos médicaments seront livrés sous 24 à 48h.
  • Retrait en pharmacie : Choisissez la pharmacie partenaire la plus proche et l'heure à laquelle vous souhaitez retirer vos médicaments (disponibles sous 2h).

Pour accéder à vos documents médicaux :

  1. Sur la page d'accueil, cliquez sur "Mes documents"
  2. Vous verrez la liste de vos ordonnances et autres documents médicaux
  3. Cliquez sur un document pour le consulter
  4. Vous pouvez également télécharger le document pour le conserver

Pour modifier les options d'accessibilité :

  1. Sur n'importe quelle page, cliquez sur le bouton "Options d'accessibilité" en haut à droite
  2. Vous pouvez alors ajuster :
    • La taille du texte (petit, moyen, grand)
    • Le contraste (normal ou élevé)
    • Réduire les animations
  3. Vos préférences seront enregistrées pour vos prochaines visites

Si vous avez oublié votre mot de passe :

  1. Sur la page de connexion, cliquez sur "Mot de passe oublié"
  2. Entrez l'adresse email associée à votre compte
  3. Vous recevrez un email avec un lien pour réinitialiser votre mot de passe
  4. Suivez les instructions dans l'email pour créer un nouveau mot de passe
